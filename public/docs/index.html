Grocery Storefront API

GraphQL API documentation for the Grocery Storefront. Build food e-commerce apps with product catalog, recipes, allergen management, nutrition tracking, and checkout.

● GraphQL ● URQL Client ● Public API

Overview

The Grocery Storefront API provides a complete set of GraphQL operations for building grocery e-commerce frontends.

Key Features Storage zones (Frozen/Chilled/Ambient) • Allergen filtering • Nutrition facts • Freshness tracking • Recipe integration • Pre-orders • Multi-currency • 8-language i18n

Authentication

Most endpoints are public and require no authentication. Optional authentication via Bearer token.

HTTP Headers
Authorization: Bearer <grocery_token>
Content-Type: application/json
Token Storage The token is stored in a browser cookie named grocery_token and automatically attached to all requests.

Endpoint

All GraphQL operations use a single endpoint.

GraphQL Endpoint
POST  https://<domain>/graphql/storefront

# Example
POST  https://chesaigon.eshoper.pro/graphql/storefront
EnvironmentURL

Query products

Fetch a paginated list of products with optional filtering and sorting.

VariableTypeRequiredDescription
channelString!YesSalon/shop slug (e.g. "chesaigon")
firstIntNoNumber of items to return
afterStringNoCursor for pagination
filterProductFilterInputNoFilter criteria (see Filtering)
sortByProductOrderNoSort order (see Sorting)
Query
query GroceryProducts(
  $channel: String!
  $first: Int
  $after: String
  $filter: ProductFilterInput
  $sortBy: ProductOrder
) {
  products(
    channel: $channel
    first: $first
    after: $after
    filter: $filter
    sortBy: $sortBy
  ) {
    edges {
      node {
        id name slug description
        thumbnail { url alt }
        allergens dietaryTags calories
        spiceLevel isAlcohol countryOfOrigin
        sellByWeight pricePerUnit unitOfMeasure
        storageZone ingredients certifications
        freshness nearestExpiry
        nutritionFacts {
          calories fat saturatedFat carbs
          sugar fiber protein salt servingSize
        }
        category { id name slug }
        pricing {
          priceRange { start { gross { amount currency } } }
          onSale
        }
        variants {
          id name sku quantityAvailable
          pricing { price { gross { amount currency } } }
          expiryTracking shelfLifeDays
          preOrder { enabled date depositPercent maxQuantity }
        }
      }
      cursor
    }
    pageInfo { hasNextPage endCursor }
    totalCount
  }
}
Variables Example
{
  "channel": "chesaigon",
  "first": 20,
  "filter": {
    "search": "pho",
    "storageZone": "AMBIENT"
  },
  "sortBy": {
    "field": "PRICE",
    "direction": "ASC"
  }
}

Mutation checkoutCreateFull

Create a new checkout session with line items.

Mutation
mutation CheckoutCreateFull($input: CheckoutCreateInput!) {
  checkoutCreateFull(input: $input) {
    checkout {
      id token email
      totalPrice { gross { amount currency } }
      availableShippingMethods {
        id name
        price { amount currency }
        minimumDeliveryDays maximumDeliveryDays
      }
      availablePaymentMethods {
        id name description provider methodType iconUrl
        fee { amount currency }
      }
    }
    errors { field message code }
  }
}
Variables Example
{
  "input": {
    "channel": "chesaigon",
    "email": "customer@example.com",
    "lines": [
      { "variantId": "variant-uuid", "quantity": 2 }
    ]
  }
}

GroceryProduct
interface GroceryProduct {
  id: string
  name: string
  slug: string
  description?: string
  thumbnail?: { url: string; alt?: string }
  category?: { id: string; name: string; slug: string }
  variants: ProductVariant[]
  price: number
  currency: string
  stockQuantity: number

  // Food-specific
  allergens?: string[]          // e.g. ["peanuts","milk","gluten"]
  dietaryTags?: string[]        // e.g. ["vegan","halal"]
  calories?: number
  spiceLevel?: number
  isAlcohol?: boolean
  countryOfOrigin?: string
  sellByWeight?: boolean
  pricePerUnit?: number
  unitOfMeasure?: string
  storageZone?: StorageZone     // FROZEN | CHILLED | AMBIENT
  ingredients?: string
  nutritionFacts?: NutritionFacts
  certifications?: string[]     // e.g. ["organic","kosher"]
  freshness?: Freshness          // FRESH | EXPIRING_SOON | LAST_CHANCE
  nearestExpiry?: string        // ISO date of nearest lot expiry
}

Filtering & Sorting

Product queries support rich filtering and sorting options.

ProductFilterInput

FieldTypeDescription
searchStringFull-text search on name and description
excludeAllergens[String]Exclude products containing these allergens
dietaryTags[String]Filter by dietary tags (vegan, halal, etc.)
certifications[String]Filter by certifications (organic, kosher)
storageZoneStorageZoneFilter by storage temperature zone

ProductOrder (Sorting)

Example
{
  "sortBy": {
    "field": "PRICE",     // NAME | PRICE | CREATED | UPDATED | RATING
    "direction": "ASC"    // ASC | DESC
  }
}
